## Build Order
1. Research design trends (use research-design skill)
2. Build layout structure + navigation
3. Build landing hero section
4. Build submission form
5. Wire Claude API for roast generation
6. Build roast results card
7. Build share functionality
8. Add Stripe paywall for PDF
9. Screenshot loop until 95%+ quality
10. Deploy to Netlify (git push → auto-deploys to www.roastmybrand.wtf)

## Screenshot Loop Rule
After every major section: screenshot → compare → list gaps → fix → repeat.
Do not move to next section until current one looks intentional and premium.

## Self-Review Checklist (run before declaring done)
- [ ] Does it look like it was made by a senior designer?
- [ ] Is the microcopy witty and on-brand?
- [ ] Do animations feel smooth, not gimmicky?
- [ ] Is mobile version tested?
- [ ] Is the CTA clear above the fold?

## Error Handling
If an approach fails twice, research an alternative before trying again.
Document what failed in a NOTES.md file.
