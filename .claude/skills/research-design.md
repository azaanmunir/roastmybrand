## Goal
Find the highest quality design references for dark premium SaaS/tool sites 
before writing a single line of UI code.

## Process
1. Search "best dark SaaS website design 2025"
2. Search "award winning tool website UI dark theme"
3. Search "Framer Motion animation examples production sites"
4. Search "viral tool landing page design inspiration"
5. Fetch top 3 results from each search
6. Extract: color palettes, typography choices, animation patterns, layout structures
7. Summarize findings into a DESIGN_BRIEF.md in the project root
8. Only after brief is written, proceed to build

## Output
A DESIGN_BRIEF.md file with:
- Recommended color palette (hex values)
- Recommended font pairing
- Key animation patterns to use
- Layout inspiration notes
- 3 microcopy examples for the roast tool tone
