# RoastMyBrand.wtf — Workspace Brain

## Project
A viral AI-powered brand roasting tool. Users submit a logo or URL. 
Claude brutally but constructively roasts their brand identity. 
Output is a shareable card. Paid tier unlocks full PDF report.

## Stack
- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion (animations)
- Claude API (roast generation)
- Stripe or LemonSqueezy (payments)
- html2canvas (card screenshot export)
- Vercel (deploy)

## Brand Personality
Brutal. Funny. Smart. Think Gordon Ramsay meets brand consultant.
The UI should feel premium-dark with sharp wit built into microcopy.

## Core Rules
- ALWAYS run /research-design skill before writing any UI code
- ALWAYS screenshot and self-review before declaring any page done
- Never use generic placeholder copy — write real witty microcopy
- Keep components modular — one file per section
- Mobile-first always

## Key Pages
1. / — Landing + submission form
2. /roast — Results page with shareable card
3. /report — Full paid PDF report
4. /pricing — Upgrade CTA

## Monetization Logic
- Free: Roast card with watermark, shareable
- Paid ($7): Full PDF report, no watermark
- Agency ($49/mo): Embed widget for their clients
